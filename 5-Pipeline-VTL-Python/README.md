# Elyra pipeline

Trevas Jupyter contains Trevas and Python Kernel.

This means we can run VTL and Python in the same environment.

Instead of running manually execution in the right direction, we also introduced [Elyra](https://elyra.readthedocs.io/en/latest/getting_started/installation.html) in the Trevas Jupyter notebook.
